Module Hierarchy

  • datetime: Fast implementation of the datetime type.
  • etl: ETL (Extract, transform, and load) is a module for python 2.4 or greater that implements ETL concepts for data import, export and also performs some operations beween import/export.
    • etl.component: Used to fetch data from any sources like files, database, etc..,transform data and store in any destinations like files, database etc.
      • etl.component.component': ETL Component.
      • etl.component.control: ETL Control components.
        • etl.component.control.data_count': Calculate total count of data while transferring.
        • etl.component.control.sleep': Puts job process in sleep.
      • etl.component.input: To extract data from the source systems.
        • etl.component.input.csv_in': To read data from csv file.
        • etl.component.input.data': This component is used to read data.
        • etl.component.input.facebook_in': This is an ETL Component that is used to read data from facebook.
        • etl.component.input.gmail_in'

        • etl.component.input.sql_in': To read data from SQL database.
        • etl.component.input.sugarcrm_in': This is an ETL Component that reads data from SugarCRM.
        • etl.component.input.vcard_in': To import data from vcard.
        • etl.component.input.xmlrpc_in': To run xmlrpc server.
      • etl.component.output: To load/store data into destination system.
        • etl.component.output.csv_out': To write data to csv file.

        • etl.component.transform.unique': To perform unique operation.
